# MassDig 2.9.0

MassDig 2.9.0 is a usability release for Minecraft 26.1.2. The mod keeps the same mining engine, but the configuration flow is now much easier to understand for regular players.

## Highlights

- New main hub with three clear modes:
  - Fast Digging: removes the client mining pause.
  - Radius Drill: mines nearby blocks while attack is held with a pickaxe.
  - AutoDig: a separate visual planner for quarries, tunnels, areas, and larger jobs.
- Radius Drill now has its own screen with simple controls first.
- Technical options such as packet limits, extra wait, skipped previews, and low-HP pauses are hidden behind Advanced.
- Built-in guide explains what to choose, how highlights work, and when to use AutoDig.
- Russian and English UI text updated for the new flow.
- Mod metadata now lists `rub1kub` as the author.

## What's New in 2.9.0

- Replaced the old dense settings list with a mode-focused hub.
- Added a dedicated Radius Drill screen.
- Added a guide screen directly inside the mod.
- Kept AutoDig as its own top-level mode instead of mixing it with fast digging or radius drilling.
- Added a one-click Server profile button to the hub.

## Compatibility

- Minecraft: 26.1.2
- Fabric Loader: 0.19.2 or newer
- Java: 25 or newer
- Fabric API: required
- Mod Menu: optional

## Artifacts

Attach these files to the GitHub release:

- `massdig-2.9.0+mc26.1.2.jar`
- `massdig-2.9.0+mc26.1.2-sources.jar`

## Notes

MassDig is a client-side quality-of-life mod. Use it only in single-player, private servers, or servers where this type of client mod is allowed.
