# MassDig 2.9.1

MassDig 2.9.1 fixes the readability of the new main menu introduced in 2.9.0.

## What's Fixed

- Replaced the overly large empty hub cards with a compact vanilla-style menu.
- Put the important mode explanations directly into visible menu rows.
- Kept the three-mode structure: Fast Digging, Radius Drill, and AutoDig.
- Kept Radius Drill and AutoDig as separate screens.
- Added clearer short labels for choosing the right mode.

## Compatibility

- Minecraft: 26.1.2
- Fabric Loader: 0.19.2 or newer
- Java: 25 or newer
- Fabric API: required
- Mod Menu: optional

## Artifacts

- `massdig-2.9.1+mc26.1.2.jar`
- `massdig-2.9.1+mc26.1.2-sources.jar`

## Notes

MassDig is a client-side quality-of-life mod. Use it only in single-player, private servers, or servers where this type of client mod is allowed.
