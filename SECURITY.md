# Security Policy

## Supported Versions

The latest release is the supported version.

## Reporting Issues

If you find a security-sensitive issue, please open a private report through GitHub Security Advisories when available, or contact the repository owner directly.

## Scope

MassDig is a client-side Minecraft mod. Security reports should focus on:

- Unsafe file handling.
- Unexpected network behavior.
- Crashes caused by malformed configuration.
- Vulnerabilities in release artifacts or build configuration.

Reports about bypassing server rules, anti-cheat systems, or moderation policies are outside the intended purpose of this project.
